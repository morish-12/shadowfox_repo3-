# 🤖 GPT-2 Language Model Exploration

> **Advanced Task** – Virtual Internship @ Shadow Fox

A comprehensive Jupyter notebook that implements and deeply analyzes **GPT-2 (small)** using Hugging Face Transformers.

---

## 🎯 Research Questions

1. How does temperature affect coherence vs. creativity?
2. Does GPT-2 maintain long-range contextual consistency?
3. How does the model handle domain-specific prompts (e.g., medical facts)?

---

## 📊 Key Findings

| Aspect | Finding |
|--------|---------|
| **Optimal Temperature** | 0.8–1.0 (best balance of fluency & novelty) |
| **Long-range Consistency** | Degrades after ~100 tokens |
| **Hallucination Risk** | Model invents side effects in medical queries |
| **Attention Pattern** | Strong self-attention & recency bias |
| **Perplexity** | ~46 on general English text |
